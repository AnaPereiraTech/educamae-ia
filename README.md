# EducaMãe IA

Aplicação desenvolvida em React + TypeScript + Vite para auxiliar mães solos atípicas no planejamento financeiro utilizando Inteligência Artificial (Google Gemini).

## Tecnologias

- React
- TypeScript
- Vite
- Tailwind CSS
- React Router
- Google Gemini API
- Recharts

## Funcionalidades

- Tema claro/escuro
- Formulário em etapas
- Salvamento no LocalStorage
- Diagnóstico financeiro com IA
- Dashboard com resumo financeiro
- Gráfico de despesas 
- Exportação em PDF
- Interface responsiva

## Como executar

```bash
npm install
npm run dev
```
